# User Stories - Learning Management System

## Learner

- **US-LRN-001**: As a learner, I want to search and filter courses so I can find relevant training quickly.
- **US-LRN-002**: As a learner, I want to resume lessons where I left off so I can learn across multiple sessions.
- **US-LRN-003**: As a learner, I want to submit quizzes and assignments and see feedback so I understand my performance.
- **US-LRN-004**: As a learner, I want to see deadlines, progress, and completion requirements so I can manage my learning plan.
- **US-LRN-005**: As a learner, I want to download or verify earned certificates when I complete courses.

## Instructor

- **US-INS-001**: As an instructor, I want to manage cohorts and schedules so I can deliver courses to the right learners.
- **US-INS-002**: As an instructor, I want to review submissions and publish grades so assessment outcomes are timely.
- **US-INS-003**: As an instructor, I want visibility into learner progress and at-risk students so I can intervene early.
- **US-INS-004**: As an instructor, I want to communicate announcements and session updates so my cohort stays informed.

## Teaching Assistant / Reviewer

- **US-REV-001**: As a reviewer, I want to moderate discussions and review assignment queues so instructor workload can be shared.
- **US-REV-002**: As a reviewer, I want rubric-guided grading so scoring remains consistent and defensible.

## Content Admin / Author

- **US-CNT-001**: As a content admin, I want to create draft courses with modules and lessons so learning content can be structured before release.
- **US-CNT-002**: As a content admin, I want versioning and publication workflows so learner-facing courses remain stable while updates are prepared.
- **US-CNT-003**: As a content admin, I want assessments and prerequisites linked to content units so completion logic is enforceable.

## Tenant Admin

- **US-TAD-001**: As a tenant admin, I want to manage users, cohorts, and course availability so my organization controls access.
- **US-TAD-002**: As a tenant admin, I want reporting on participation, progress, and outcomes so I can evaluate training effectiveness.
- **US-TAD-003**: As a tenant admin, I want tenant-scoped integrations and policies so the LMS fits organizational needs.

## Platform Admin

- **US-PAD-001**: As a platform admin, I want to manage tenant lifecycles, integrations, and operational controls so the service stays reliable and compliant.
- **US-PAD-002**: As a platform admin, I want complete audit logs and observability so incidents and compliance reviews are manageable.
